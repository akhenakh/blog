Les applications essentielles pour Mac

Share

Voici la liste des applications indispensables pour votre Mac, la plupart sont gratuites, les applications payantes sont signalées par €€€.

Nouvelle liste !

Après une installation ou une réinstallation de votre Mac, vous trouverez cette liste peut être aussi utile que moi.

Firefox Le browser web libre et gratuit

OpenOffice 3 Enfin une alternative viable et gratuite à Microsoft Office, associé aux templates

VLC Le player vidéo multiformat

Windows Media Components for QuickTime Les CODECS windows media pour permettre la lecture dans Quicktime

DIVX Les CODECS DivX pour permettre la lecture dans Quicktime

Perian Un composant Quicktime qui permet la lecture de tous les formats, AVI, MKV … dans quicktime, permet aussi la lecture dans FrontRow (le menu via la télécommande)

AdiumX Le logiciel me messagerie instanté multi comptes: AIM, MSN, ICQ, Yahoo, Facebook …

Skype Téléphoner par internet

Colloquy Un client IRC

Xlite Un client SIP pour téléphoner via votre ordinateur, avec par exemple une ligne freephonie

Chax Un plugin Ichat, qui permet de le configurer plus finement

AMSN Un client MSN qui supporte la vidéo, les wizzs et autres saletés du protocole MSN parfois nécessaire …

Burn Un logiciel de gravure CD, DVD

The Unarchiver Le décompacteur de fichier multiformats, ZIP, RAR, 7z …

SmartReporter Vous informe, dans la barre Mac Os, de l’état de vos disques durs en lisant les informations SMART (peut éviter une catastrophe)

Google Earth La terre et les étoiles en 3D, le logiciel géographique de Google

Gmail notifier Vous informe quand vous recevez un mail sur votre boite Gmail

Google Videochat La video et le son dans Gmail, (traverse bien les firewalls)

Growl est un systeme de notification par popup qui fonctionne avec une centaine applications: Mail , iTunes, adiumx …

Last.fm le client Last.fm, qui analyse la musique que vous écoutez et vous propose de la musique en rapport avec vos gouts

Thunderbird si vous n’appréciez pas Mail, un client mail développé par la Mozilla Fondation (éditeur de Firefox), supporte des plugins comme Enigmail qui permet de crypter/decrypter vos mails PGP/GPG

Istats Menu Température de votre ordinateur, mémoire utilisée … tout cela indiqué dans la barre Mac Os

Appfresh Un outil pour mettre à jour tous les logiciels de cette page automatiquement

Virtualbox Windows, linux, FreeBSD dans votre mac, gratuitement, un concurrent sérieux à Vmware, rapide et très stable

Gimp Une alternative gratuite à Photoshop

Audacity Un éditeur de son, permet l’export en MP3 via un plugin lui aussi gratuit

Adobe DNG converter et Cameraw Pour convertir le format RAW de votre appareil photo en DNG lisible par photoshop, comme le LX3 par exemple

Tomato Torrent Petit client bittorrent simple et efficace, pour les gros utilisateurs le fabuleux Utorrent enfin sur Mac.

OmniFocus €€€ gestionnaire de tache multifoncion, je ne sais plus rien faire sans lui

Chmox Pour lires les docs en .CHM

Waterroof Gére le firewall de Mac OS X (ipfw) pour les experts

Macfuse Est un procédé qui permet de voir des fichiers et répertoires virtuels, par exemple un filsystem distant ssh comme étant local à votre mac, cela permet aussi de lire et écrire du ntfs sur votre mac via NTFS-3G.

Skitch Est un logiciel de capture d’image très simple et qui permet de facilement de partager ces captures sur le net

Dropbox Un système d’échange de fichier privée/publique par le web avec une application native, vous permettant de partager très facilement des fichiers

Pixelpipe De multiples plugins pour iPhoto, lightroom … qui vous permettront d’envoyer vos images sur plusieurs réseaux sociaux et autres systèmes de galerie comme Facebook, Flickr, Picasa …

Google Quick Search Un remplaçant simple du laborieux spotlight, vous permet de lancer vos applications, de trouver un contact ou de faire une recherche sur Google, ressemble en beaucoup plus simple à Quicksilver projet génial mais qui semble abandonné

Pour l’Unixien:

iTerm Le remplacant de terminal

X11 mise à jour Le package X11 est fournit dans le CD d’installation Mac OsX mais c’est une vieille version, > 2.3.2 est recommandé pour Gimp par exemple

voir macports plus bas

Pour le développeur:

Extensions pour Firefox Firebug et Web Developper pour le développeur Web

Xcode L’outil de développement Apple, un IDE, gcc sauce apple, même si vous ne développez pas , il est utile pour compiler des programmes, via macports par exemple.

Netbeans L’éditeur Java, mais aussi Python, Ruby, C/C++, Android … (N’oubliez pas d’activer Java 6 dans Applications/Utilitaires/Java)

SmartSVN Un front end SVN pour gérer votre repository

PgAdmin Un logiciel d’administration de votre base Postgresql, permet aussi de voir et modifier vos données

TextMate €€€ Un éditeur texte multifonction, alternative en gratuit:Jedit ou TextWrangler

Transmit €€€ Client FTP/SFTP vous pouvez lui préférer Cyberduck Un client FTP, pour télécharger des fichiers, gratuit

Onyx Un outil de personnalisation de Mac Os X, qui donne notamment accès à des paramètres cachés, pour l’utilisateur expert

Developper Color Picker Un outil de sélection de couleur pour le développeur Cocoa et Web

Macports le gestionnaire de paquet pour logiciel opensource, apache, php, python, le monde opensource linux dans votre mac
