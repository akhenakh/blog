Appcode shortcuts

^ up and ^ down : navigate methods

⇧ ⌘ up and ⇧ ⌘ down : switch from .h to .m

⇧ ⌘ ⌫ : go to last edit

⇧ esc : hide console/bottom view

⌥ ⌘ L : reindent code (^ i in xcode)

^ r : Run (⌘ + r in xcode)

^ tab : navigate files

⌘ o : open class

⌘ ⇧ o : open files

^ ⇧ space : smart completion (completing blocks arguments)

⇥ : next parameters (⌘/ in xcode)

⌘ F9 : build (⌘ b in xcode)

^ o : override methods

^ i : implements methods

⌘ n : generate code

⌘ ⌥ F7 : show usages

F1 : Fast documentation

⇧ F1 : Apple documentation
